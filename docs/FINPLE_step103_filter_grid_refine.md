# FINPLE Step 103-9C — Filter Grid Refine

## 작업 개요

스크리너의 시장 드롭다운을 별도 위치에서 제거하고, 필터 그리드 안으로 통합했습니다.

## 변경 사항

```text
- 필터를 4개 드롭다운으로 정리
- 1차 시장
- 2차 자산군
- 3차 투자 스타일
- 4차 위험도
- 자산 성격 필터는 별도 분리하지 않음
- 투자 스타일 문구를 배당형 / 성장형 / 방어형 / 공격형으로 정리
- 초보자, 레버리지/인버스는 기존 표현 유지
```

## 현재 분류 기준

```text
시장 → 자산군 → 투자 스타일 → 위험도 → 검색어
```

## 메모

채권, 원자재, 부동산/리츠, 가상화폐는 엄밀히는 자산 성격에 가깝지만, 현 단계에서는 필터 수를 4개로 유지하기 위해 투자 스타일 드롭다운 안에 포함했습니다.
