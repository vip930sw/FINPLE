# FINPLE Step 103-10 — KR Candidate Add

## 작업 개요

한국 스크리너 후보를 포트폴리오에 담을 수 있도록 1차 연결했습니다.

## 변경 사항

```text
- 한국 후보 카드의 준비 중 상태를 추가 가능 상태로 변경
- 전체 시장에서도 미국 후보와 한국 후보 모두 추가 가능
- 한국 후보 추가 시 market: KR 유지
- 한국 후보 추가 시 quoteCurrency: KRW 유지
- 한국 후보 추가 시 assetType 유지
- 한국 후보는 priceMode를 manual로 시작
- 미국 후보 추가 기존 동작 유지
```

## 의도적으로 제외한 항목

```text
- 한국 현재가 API 조회
- 한국 CAGR / BETA / MDD / 배당률 자동 계산
- 한국 전용 시뮬레이터 화면 연결
- 한국/미국 혼합 포트폴리오 계산 보정
```
