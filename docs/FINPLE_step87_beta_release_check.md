# FINPLE Step 87. 베타 공개 버전 확정 / 최종 배포 체크

작성일: 2026-05-09

## 1. 현재 상태 요약

FINPLE Portfolio Lab은 현재 베타 공개 전 최종 확인 단계입니다.

현재 완료된 주요 항목은 다음과 같습니다.

| 구분 | 상태 | 메모 |
|---|---:|---|
| 도메인 연결 | 완료 | https://finple.co.kr 정상 연결 |
| 관리자 전용 주소 | 완료 | /admin 직접 접속 |
| 법적 문서 | 완료 | 개인정보처리방침 / 이용약관 / 투자 유의사항 |
| 문의사항 | 완료 | 작성 / 관리자 조회 / 상태 변경 확인 |
| 포트폴리오 저장 | 완료 | 서버 저장 / 불러오기 확인 |
| Free 제한 | 완료 | 포트폴리오 1개 기준 적용 |
| 모바일 핵심 화면 | 완료 | 주요 화면 확인 완료 |
| 운영용 문구 정리 | 완료 | 개발자용 문구를 체험/베타 문구로 정리 |

---

## 2. 베타 공개 주소

### 일반 사용자 주소

```text
https://finple.co.kr
```

### 관리자 주소

```text
https://finple.co.kr/admin
```

관리자 주소는 일반 사용자에게 공개하지 않습니다.

---

## 3. 공개 전 최종 확인 순서

### 3-1. 도메인 / 라우팅

아래 주소를 직접 접속해서 확인합니다.

```text
https://finple.co.kr/
https://finple.co.kr/admin
https://finple.co.kr/privacy
https://finple.co.kr/terms
https://finple.co.kr/disclaimer
```

확인 기준:

- 직접 접속 가능
- 각 문서 페이지에서 홈 이동 시 주소가 / 로 정리됨
- finple-api.vercel.app은 finple.co.kr로 리다이렉트됨
- www.finple.co.kr은 보류 상태여도 메인 도메인 운영에는 영향 없음

---

### 3-2. 일반 사용자 로그인

확인 순서:

1. https://finple.co.kr 접속
2. 로그인 클릭
3. 체험 계정으로 시작
4. MY PAGE 이동 확인
5. 계정 상태 확인

확인 기준:

- 일반 로그인 화면에 관리자 카드가 보이지 않음
- 체험 계정 문구가 자연스러움
- MY PAGE 접근 가능

---

### 3-3. 관리자 로그인

확인 순서:

1. https://finple.co.kr/admin 접속
2. 관리자 토큰 입력
3. MY PAGE 이동
4. 관리자 문의 조회 패널 확인
5. 관리자 모드 해제
6. 관리자 패널 숨김 확인

확인 기준:

- /admin에서만 관리자 로그인 가능
- 관리자 토큰 없이 관리자 패널 미노출
- 관리자 모드 해제 후 패널 숨김

---

### 3-4. 포트폴리오 저장 / 불러오기

확인 순서:

1. 시뮬레이터 접속
2. 자산 수량 또는 조건 변경
3. 서버 저장
4. 새로고침
5. 서버 불러오기
6. 저장 시점 값으로 복구되는지 확인

확인 기준:

- Free 플랜에서 포트폴리오 1개만 표시
- 서버 불러오기 후 중복 포트폴리오가 생성되지 않음
- 새로고침 후에도 1개 유지

---

### 3-5. 문의사항

확인 순서:

1. 문의사항 페이지 접속
2. 테스트 문의 1건 작성
3. /admin 접속
4. 문의 목록 불러오기
5. 상태를 확인 중으로 변경
6. 상태를 종료로 변경

확인 기준:

- 신규 문의 정상 등록
- 관리자 화면에서 정상 조회
- 상태 변경 정상 반영
- 종료 필터에서 확인 가능

---

### 3-6. 법적 문서 / 고지 문구

확인 항목:

- 개인정보처리방침
- 이용약관
- 투자 유의사항
- 요금제 하단 투자 유의 문구
- 문의사항 개인정보 수집 안내

확인 기준:

- 문구가 화면 밖으로 밀리지 않음
- 모바일에서도 자연스럽게 줄바꿈됨
- 투자 자문 / 수익 보장 오해를 줄이는 문구가 표시됨

---

### 3-7. 모바일 핵심 화면

확인 대상:

- 홈
- 로그인
- /admin
- 요금제
- 문의사항
- 시뮬레이터
- 포트폴리오 저장 영역

확인 기준:

- 주요 버튼 클릭 가능
- 카드 / 문구가 화면 밖으로 과도하게 밀리지 않음
- 시뮬레이터 표는 모바일에서 완벽한 편집보다는 조회 가능성을 기준으로 판단
- 자세한 투자 입력은 PC 사용 권장

---

## 4. 베타 공개 가능 기준

아래 조건이 모두 충족되면 베타 공개 가능으로 판단합니다.

| 항목 | 기준 |
|---|---|
| 접속 | finple.co.kr 정상 접속 |
| 저장 | 포트폴리오 서버 저장 / 불러오기 정상 |
| 문의 | 문의 작성 / 관리자 조회 정상 |
| 관리자 | /admin 분리 및 토큰 검증 정상 |
| 고지 | 투자 유의 문구 노출 |
| 모바일 | 핵심 화면 사용 가능 |
| 요금제 | Free / Personal / Pro 문구 표시 |
| 데이터 | 테스트 문의 종료 처리 |

## 5. 공개 전 HOLD 항목

베타 공개 시 아래 항목은 아직 제공하지 않는 것으로 명확히 둡니다.

| 항목 | 상태 | 비고 |
|---|---:|---|
| 실제 이메일 회원가입 | HOLD | 체험 계정으로 대체 |
| 소셜 로그인 | HOLD | 추후 연동 |
| 실제 결제 / 구독 | HOLD | 요금제 선택만 표시 |
| Pro 플랜 | HOLD | 준비 중 |
| PDF 리포트 고도화 | HOLD | 추후 개선 |
| 포트폴리오 삭제 / 복구 고도화 | HOLD | 기본 삭제만 유지 |
| 모바일 시뮬레이터 고도화 | HOLD | PC 사용 권장 |
| API 조회량 과금 연동 | HOLD | 추후 결제 연동 후 처리 |

---

## 6. 공개 시 권장 안내 문구

베타 공개 시 서비스 소개 또는 공지에 아래 문구를 사용할 수 있습니다.

```text
FINPLE Portfolio Lab은 포트폴리오의 장기 예상 흐름과 위험 지표를 확인할 수 있는 베타 서비스입니다.

현재는 체험 계정 기반으로 제공되며, 일부 기능은 브라우저 저장 또는 제한된 서버 저장 방식으로 운영됩니다. 분석 결과는 투자 판단을 돕기 위한 참고자료이며, 특정 금융상품의 매수·매도 추천이나 수익 보장을 의미하지 않습니다.
```

---

## 7. 공개 후 첫 1주일 확인 항목

베타 공개 후 1주일 동안 아래 항목을 우선 관찰합니다.

- 문의사항 접수 여부
- 서버 저장 실패 여부
- Alpha Vantage API 호출 제한 빈도
- 모바일 화면 불편 의견
- 요금제 문구 오해 여부
- 개인정보 / 투자 유의사항 관련 문의
- Render free instance 지연으로 인한 체감 속도 문제
