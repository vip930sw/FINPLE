# FINPLE 환불정책 법적 검토 메모

작성일: 2026-06-13  
대상 서비스: FINPLE Personal 월 구독형 웹앱  
성격: 내부 운영 검토용 문서 / 법률 자문 대체 아님

---

## 1. 문서 목적

이 문서는 FINPLE의 Personal 플랜 및 향후 유료 구독 운영과 관련하여, 환불정책을 어떻게 설계해야 법적·심사상 리스크를 줄일 수 있는지 정리한 내부 검토 문서이다.

핵심 결론은 다음과 같다.

```text
FINPLE은 “전면 환불불가 서비스”로 설계하면 안 된다.
다만 결제 즉시 유료 기능이 제공되는 디지털 구독형 서비스이므로,
유료 기능 이용이 시작된 경우 환불을 제한하는 구조는 가능하다.
```

따라서 FINPLE의 환불정책은 아래 방향이 가장 안전하다.

```text
1. 결제 후 7일 이내 + 유료 기능 미사용 → 전액 환불
2. 유료 기능 사용 후 → 환불 제한 가능
3. 구독 해지 → 언제든 가능
4. 해지 후 다음 결제일부터 자동결제 중단
5. 중복결제·오결제·미제공·회사 귀책 장애 → 환불 또는 보상
6. 결제 전 고지와 필수 동의 체크박스 → 필수
```

---

## 2. FINPLE 서비스 성격

FINPLE은 투자자문 서비스가 아니라, 사용자가 직접 입력하거나 선택한 포트폴리오를 기준으로 장기 성과, 위험지표, 배당, 실질가치 등을 계산해보는 구독형 포트폴리오 분석 SaaS이다.

유료화 포인트는 다음 기능이다.

```text
- 서버 저장
- PDF 리포트
- 포트폴리오 비교
- 조회한도 확대
- 데이터 편의성
- MY PAGE 관리
- 결제내역 / 문의내역 관리
```

환불정책도 이 서비스 성격에 맞춰 “투자 성과 불만족”이 아니라 “디지털 서비스 제공 여부와 이용 개시 여부”를 기준으로 설계해야 한다.

## 3. 관련 법적 프레임

환불정책 검토 시 최소한 아래 법적 틀을 함께 봐야 한다.

```text
1. 전자상거래법
   - 7일 청약철회
   - 디지털콘텐츠·용역 제공 개시 시 청약철회 제한 가능
   - 청약철회 제한 조건의 명확한 표시 필요

2. 방문판매법상 계속거래

3. 콘텐츠이용자 보호지침

5. 소비자분쟁해결기준 및 공정위 사례
   - 실제 소비자 분쟁 시 참고 기준
```

## 4. 전자상거래법상 기본 구조

전자상거래법상 소비자는 원칙적으로 계약내용에 관한 서면을 받은 날부터 7일 이내 청약철회를 할 수 있다.

다만 다음 경우에는 청약철회 제한 사유가 될 수 있다.

```text
- 용역 제공이 개시된 경우
- 디지털콘텐츠 제공이 개시된 경우
- 소비자가 이를 명확히 인지하고 동의한 경우
- 시험 사용 또는 체험 수단이 제공된 경우
```

FINPLE에서 Personal 기능 제공 개시로 볼 수 있는 행위는 다음과 같다.

```text
- 서버 저장 사용
- PDF 리포트 생성
- 유료 조회한도 사용
- 포트폴리오 비교 기능 사용
- Personal 전용 MY PAGE 기능 사용
- 유료 결제수단 및 구독 상태 관리 기능 사용
```

따라서 FINPLE은 결제 직후 다음 내용을 분명히 고지해야 한다.

```text
결제 완료 즉시 Personal 기능이 활성화되며,
서버 저장, PDF 리포트 생성, 유료 조회한도 사용 등 유료 기능을 사용한 경우
디지털 서비스 제공이 개시된 것으로 보아 환불이 제한될 수 있습니다.
```

## 5. Free 플랜의 역할

FINPLE은 Free 플랜을 단순 무료버전이 아니라 “시험 사용 또는 체험 수단”으로 설계해야 한다.

Free 플랜의 목적은 다음이다.

```text
- 서비스 목적 이해
- 포트폴리오 시뮬레이션 체험
- 투자 MBTI 체험
- 제한된 저장 또는 브라우저 저장 체험
- Personal 기능 필요성 확인
```

이 구조는 향후 청약철회 제한을 설명할 때도 도움이 된다.

```text
사용자는 결제 전 Free 플랜을 통해 서비스의 주요 구조를 미리 체험할 수 있음.
Personal은 저장, 리포트, 비교, 조회한도 확대 등 확장 기능을 제공하는 유료 구독임.
```

## 6. 월 구독과 계속거래 이슈

FINPLE Personal은 월 9,900원 정기결제 구조이므로, 1회성 디지털콘텐츠 판매보다 계속거래 성격이 생길 수 있다.

따라서 환불정책과 별개로, 구독 해지는 반드시 쉽게 제공해야 한다.

```text
- 구독은 언제든지 해지 가능
- 해지 시 다음 결제일부터 자동결제 중단
- 이미 결제된 이용기간은 만료일까지 사용 가능
- 해지 버튼 또는 해지 요청 경로를 숨기지 않음
```

주의할 점은 다음이다.

```text
“해지 불가”는 위험하다.
“환불 제한”은 조건부로 가능하다.
```

## 7. 위험한 문구

아래 문구는 약관규제법 및 소비자보호 관점에서 위험하므로 사용하지 않는다.

```text
결제 후 어떠한 경우에도 환불되지 않습니다.
웹앱 특성상 모든 환불은 불가합니다.
구독상품은 환불이 불가능합니다.
단순 변심을 포함한 모든 환불 요청은 처리하지 않습니다.
이미 결제된 월 이용료는 어떤 경우에도 환불하지 않습니다.
```

위 문구는 소비자의 청약철회권, 해지권, 원상회복 청구권을 부당하게 제한하는 것으로 보일 수 있다.

## 8. 권장 환불정책 구조

FINPLE의 권장 환불정책은 다음과 같다.

| 상황 | 권장 처리 |
|---|---|
| 결제 후 7일 이내 + Personal 유료 기능 미사용 | 전액 환불 |
| 결제 후 7일 이내 + Personal 유료 기능 사용 | 환불 제한 가능 |
| 결제 후 7일 경과 | 원칙적으로 환불 제한 |
| 중복결제 | 전액 환불 |
| 오결제 | 전액 환불 |
| 결제 후 Personal 미활성화 | 전액 환불 또는 정상 제공 |
| 회사 귀책 중대한 장애 | 환불 또는 기간연장 |
| 표시·광고와 실제 제공 내용이 현저히 다름 | 환불 또는 보완 |
| 단순 변심 | 이용내역에 따라 제한 가능 |
| 구독 해지 | 언제든 가능 |
| 해지 후 남은 기간 | 만료일까지 이용 |
| 다음 결제 | 해지 후 중단 |

## 9. 결제 화면 필수 동의 체크박스

FINPLE 결제 화면에는 아래 체크박스를 두는 것이 좋다.

```text
[필수] Personal 플랜은 결제 완료 즉시 유료 기능이 활성화되며, 서버 저장·PDF 리포트·조회한도 확대 등 유료 기능을 사용한 경우 환불이 제한될 수 있음을 확인했습니다.

[필수] 결제 후 7일 이내이고 유료 기능 이용내역이 없는 경우에는 환불 요청이 가능하다는 점을 확인했습니다.

[필수] 구독은 언제든지 해지할 수 있으며, 해지 시 다음 결제일부터 자동결제가 중단된다는 점을 확인했습니다.

[필수] 환불정책, 이용약관, 개인정보처리방침을 확인했습니다.
```

결제 버튼 근처에는 아래 요약 문구를 함께 표시한다.

```text
월 9,900원 / 매월 자동결제
결제 즉시 Personal 기능 활성화
해지 시 다음 결제일부터 자동결제 중단
```

## 10. 환불정책 문구 초안

아래 문구를 `/refund` 페이지 또는 이용약관의 결제·환불 조항에 반영할 수 있다.

```text
FINPLE Personal 플랜은 결제 완료 즉시 유료 기능이 활성화되는 디지털 구독형 서비스입니다.

이용자는 결제 후 7일 이내에 환불을 요청할 수 있습니다. 다만 해당 기간 내에 서버 저장, PDF 리포트 생성, 유료 조회한도 사용, 포트폴리오 비교 등 Personal 유료 기능을 사용한 경우에는 디지털 서비스 제공이 개시된 것으로 보아 환불이 제한될 수 있습니다.

결제 후 7일 이내이고 Personal 유료 기능 이용내역이 없는 경우에는 전액 환불합니다.

중복결제, 오결제, 결제 후 서비스 미제공, 회사 귀책으로 인한 중대한 이용 장애, 표시·광고 또는 약정한 내용과 다른 서비스 제공이 확인되는 경우에는 이용 여부와 관계없이 환불 또는 보상 조치를 제공합니다.

구독은 언제든지 해지할 수 있으며, 해지 시 다음 결제일부터 자동결제가 중단됩니다. 이미 결제된 이용기간은 만료일까지 사용할 수 있습니다.

이미 제공이 개시된 월 이용료는 원칙적으로 일할 환불하지 않습니다. 다만 관련 법령상 환불이 필요한 경우 또는 회사 귀책 사유가 확인되는 경우에는 예외적으로 환불할 수 있습니다.
```

## 11. 운영 체크리스트

환불정책을 실제 운영에 반영하기 전 아래를 확인한다.

```text
- /refund 페이지에 최신 환불정책 문구 반영
- /terms 이용약관 내 결제·환불 조항과 내용 일치
- /pricing 결제 전 고지 문구 반영
- Personal 결제 버튼 근처 자동결제 금액·주기 표시
- 환불 제한 조건을 결제 전 명확히 고지
- 필수 동의 체크박스 구현
- 구독 해지 경로 제공
- MY PAGE에서 다음 결제일 / 해지 상태 확인 가능
- 결제내역 페이지에서 결제일, 금액, 영수증 링크 표시
- 중복결제·오결제·미제공·장애 환불 처리 절차 내부화
```

## 12. 토스페이먼츠 / 네이버 검수 관점 메모

환불정책은 단순 법적 리스크뿐 아니라 PG 심사와 소셜 로그인 검수에도 영향을 줄 수 있다.

검수 대응 시 강조할 포인트는 다음이다.

```text
- FINPLE은 특정 금융상품의 매수·매도·보유를 권유하지 않는 분석 도구임.
- 유료 서비스는 저장, 리포트, 비교, 조회한도 확대 등 웹서비스 이용 권한임.
- 결제 전 금액, 결제주기, 자동결제 여부, 환불정책을 고지함.
- 구독은 언제든지 해지 가능하며 다음 결제일부터 자동결제가 중단됨.
- 중복결제, 오결제, 미제공, 회사 귀책 장애는 환불 또는 보상함.
```

기존 PR #145에서 `/refund` 라우터와 푸터 환불정책 링크가 추가되었으므로, 본 문서는 그 정책의 법적·운영 기준을 보완하는 내부 문서로 관리한다.

## 13. 최종 운영 원칙

```text
FINPLE은 환불불가 서비스가 아니다.
FINPLE은 조건부 환불 제한이 가능한 디지털 구독형 서비스다.

소비자가 결제 전 충분히 확인하고,
결제 후 유료 기능 이용이 시작된 경우에는 환불을 제한할 수 있다.

다만 유료 기능을 사용하지 않은 7일 이내 결제,
중복결제, 오결제, 미제공, 회사 귀책 장애는 환불 또는 보상한다.
```
