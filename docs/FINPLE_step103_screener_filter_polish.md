# FINPLE Step 103-9B — Screener Filter Polish

## 작업 개요

스크리너 필터 UX 보정 작업입니다.

## 변경 사항

```text
- 검색 input 확장 및 검색 버튼 우측 정렬
- 자산군 전체 문구를 전체 자산군으로 변경
- 스타일 전체 문구를 전체 스타일로 변경
- 원자재 필터에서 금융이 섞이지 않도록 판별 로직 보정
- 2차 스타일에 채권 추가
- 2차 스타일에 가상화폐 추가
- 시장 선택을 드롭다운으로 전환
```

## 시장 표현 기준

국가보다 시장이라는 표현을 사용합니다.

```text
전체 시장
미국 시장
한국 시장
```

이유는 거래소, 통화, 티커 체계, 데이터 소스가 달라지는 분류이기 때문입니다.

## 분류 검토

현재 구조는 아래 기준입니다.

```text
시장 → 자산군 → 스타일 → 위험도 → 검색어
```

- 시장: 미국 시장 / 한국 시장
- 자산군: ETF / 개별주
- 스타일: 배당, 성장, 방어, 원자재, 채권, 리츠 등
- 위험도: 낮음~중간, 중간, 높음 등

향후 후보군이 더 늘어나면 `상품 종류` 또는 `자산 성격` 필터를 별도 단계로 분리할 수 있습니다.
