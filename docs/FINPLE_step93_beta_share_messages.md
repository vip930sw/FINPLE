# FINPLE Step 93. 소수 사용자 공유용 안내 메시지

## 1. 목적

FINPLE 베타 버전을 지인, 초기 테스터, 투자 관심 사용자에게 공유할 때 사용할 안내 문구를 정리한다.

현재 FINPLE은 정식 유료 서비스가 아니라 베타 운영 단계이며, 기능 안정성, 사용 흐름, 포트폴리오 저장/불러오기, 문의사항 접수, 투자 유의 문구 노출 상태를 확인하는 것이 목적이다.

---

## 2. 공유 기본 원칙

- 너무 넓게 공개하지 않고 소수 사용자에게 먼저 공유한다.
- 투자 추천 서비스가 아니라 포트폴리오 분석 도구임을 명확히 알린다.
- 베타 운영 중이므로 오류와 개선 의견을 적극적으로 요청한다.
- 개인정보, 투자 판단, 실제 매수·매도 결정에 대한 유의사항을 함께 안내한다.
- 사용자는 가능하면 PC에서 시뮬레이터를 먼저 테스트하도록 안내한다.

---

## 3. 짧은 공유 메시지

```text
FINPLE 베타 버전을 열어봤습니다.

포트폴리오를 입력하면 장기 예상 성과, 실질가치, 배당, 위험 지표를 확인할 수 있는 투자 분석 도구입니다.

아직 베타 운영 단계라 일부 기능과 데이터는 테스트 중입니다.
사용해보시고 불편한 점이나 필요한 기능이 있으면 문의사항으로 남겨주시면 감사하겠습니다.

https://finple.co.kr
```

---

## 4. 지인/테스터용 상세 메시지

```text
안녕하세요.

제가 준비 중인 포트폴리오 분석 웹앱 FINPLE의 베타 버전을 공유드립니다.

FINPLE은 사용자가 보유 자산, 월 투자금, CAGR, MDD, 배당률 등을 입력하면 장기 포트폴리오 성과와 실질가치 흐름을 확인할 수 있는 분석 도구입니다.

현재 확인 가능한 기능은 다음과 같습니다.

1. 포트폴리오 시뮬레이터
2. 자산 스크리너
3. 포트폴리오 서버 저장 / 불러오기
4. 상세 분석 리포트
5. 문의사항 접수
6. 개인정보처리방침 / 이용약관 / 투자 유의사항

현재는 베타 운영 단계이므로 실제 투자 판단 전에는 반드시 증권사, 운용사, 공식 자료를 함께 확인해 주세요.
FINPLE은 특정 종목이나 ETF의 매수·매도를 추천하거나 수익을 보장하는 서비스가 아닙니다.

사용해보시고 아래 내용을 알려주시면 큰 도움이 됩니다.

- 사용 흐름이 자연스러운지
- 포트폴리오 입력이 어렵지 않은지
- 분석 결과가 이해하기 쉬운지
- 모바일에서 불편한 화면이 있는지
- 추가되면 좋을 기능이 있는지
- 오류가 발생한 부분이 있는지

베타 주소:
https://finple.co.kr

감사합니다.
```

## 5. SNS / 커뮤니티용 짧은 문구

```text
포트폴리오 분석 웹앱 FINPLE 베타 버전을 공개했습니다.

보유 자산과 투자 조건을 입력하면 장기 성과, 실질가치, 배당, 위험 지표를 확인할 수 있습니다.

현재는 베타 운영 단계이며, 투자 추천이나 수익 보장 서비스는 아닙니다.
사용 후 피드백을 남겨주시면 개선에 반영하겠습니다.

https://finple.co.kr
```

## 6. 테스터에게 요청할 피드백 질문

### 필수 질문

1. 처음 접속했을 때 어떤 서비스인지 이해되었나요?
2. 포트폴리오 시뮬레이터 사용 흐름이 자연스러웠나요?
3. 자산 입력, 저장, 불러오기 과정에서 막히는 부분이 있었나요?
4. 분석 결과의 숫자와 차트가 이해하기 쉬웠나요?
5. 모바일에서 사용하기 불편한 화면이 있었나요?
6. 투자 유의사항과 베타 안내 문구가 충분히 보였나요?
7. 추가되면 좋을 기능이 있나요?

### 오류 신고 시 받고 싶은 정보

- 접속 기기: PC / 모바일
- 브라우저: Chrome / Edge / Safari 등
- 오류가 발생한 화면
- 어떤 버튼을 눌렀는지
- 오류 문구 또는 캡처
- 다시 시도했을 때도 반복되는지

## 7. 공유 전 최종 확인

공유 전 아래 항목을 다시 확인한다.

- [ ] https://finple.co.kr 접속 가능
- [ ] 로그인 → 체험 계정 시작 가능
- [ ] 포트폴리오 1개 제한 유지
- [ ] 서버 저장 / 불러오기 정상
- [ ] 문의사항 작성 정상
- [ ] /admin 관리자 조회 정상
- [ ] 개인정보처리방침 / 이용약관 / 투자 유의사항 접근 가능
- [ ] 모바일 홈 / 로그인 / 문의사항 / 요금제 화면 정상
- [ ] 베타 안내 문구 노출
- [ ] 투자 유의 문구 노출

## 8. 공개 범위 제안

### 1차

- 가까운 지인 3~5명
- 투자 경험이 있는 사용자
- PC 사용이 가능한 사용자

### 2차

- 투자 커뮤니티 소규모 공유
- 건축/직장 동료 중 투자 관심자
- 피드백을 자세히 줄 수 있는 사용자

### 3차

- SNS 또는 유튜브 채널 연동
- 사용법 콘텐츠 제작
- 정식 베타 안내 페이지 추가

---

## 9. 공개 후 첫 1주일 확인 항목

- 문의사항이 정상적으로 쌓이는지
- 관리자 조회가 문제 없는지
- 저장/불러오기 오류가 없는지
- API 조회 제한 관련 문의가 있는지
- 모바일 불편 피드백이 반복되는지
- 투자 추천으로 오해하는 사용자가 있는지
- 요금제 문구가 과도하거나 부족하지 않은지
