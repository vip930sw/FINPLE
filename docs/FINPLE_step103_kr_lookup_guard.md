# FINPLE Step 103-11B — KR Lookup Guard

## 작업 개요

한국 후보 자산이 포트폴리오에 추가된 뒤, 아직 연결되지 않은 한국 현재가 API 대신 미국/Alpha Vantage 조회 흐름으로 들어가는 것을 방지했습니다.

## 변경 사항

```text
- 6자리 한국형 티커를 한국 자산으로 감지
- 한국 자산은 현재가 자동 조회 전 단계로 처리
- 한국 자산 조회 시 미국 조회 API 호출 전 차단
- 한국 자산 조회 시 수동 입력 필요 안내 메시지 표시
- 미국 자산 조회 기존 동작 유지
```

## 현재 동작

```text
미국 자산
- 기존 Backend API / Alpha Vantage 흐름 유지

한국 자산
- 한국 API 연결 전까지 자동 조회를 수행하지 않음
- 현재가, CAGR, BETA, MDD, 배당률은 수동값 확인 필요
- 기존 입력값은 유지
```

## 의도적으로 제외한 항목

```text
- 한국 현재가 API 연결
- 한국 장기 가격 데이터 연결
- 한국 배당 데이터 연결
- 한국 자산 상태 배지 세부 UI 개선
```
