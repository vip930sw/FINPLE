# FINPLE Beta Changelog

## 문서 목적

이 문서는 FINPLE 베타 버전의 공개 상태, 주요 변경 사항, 보류 기능, 다음 개선 방향을 기록하기 위한 변경이력 문서입니다.

베타 운영 중에는 기능을 무리하게 추가하기보다, 사용자 피드백을 기준으로 우선순위를 정리하고 버전 단위로 개선합니다.

## Beta v0.1 요약

Beta v0.1은 FINPLE의 1차 베타 공개 버전입니다.

핵심 목적은 사용자가 실제로 포트폴리오를 입력하고, 장기 성과와 위험 지표를 확인하며, 서버 저장과 문의 기능까지 정상적으로 사용할 수 있는지 검증하는 것입니다.

### 주요 기능

- 홈 화면 및 서비스 소개
- 체험 계정 기반 로그인
- 포트폴리오 시뮬레이터
- CAGR, MDD, BETA, 배당률 기반 장기 성과 계산
- 실질 평가금액, 누적 납입금, 수익금, 배당금 차트
- Free 플랜 기준 포트폴리오 1개 제한
- 서버 저장 / 서버 불러오기
- 문의사항 작성
- 관리자 전용 주소 `/admin`
- 관리자 문의 조회 및 상태 변경
- 개인정보처리방침
- 이용약관
- 투자 유의사항
- 베타 안내 문구
- 모바일 핵심 화면 대응

## Beta v0.2 후보

사용자 피드백을 받은 뒤 아래 항목 중 우선순위를 정합니다.

### P0 후보

즉시 수정이 필요한 오류입니다.

- 접속 불가
- 로그인 불가
- 서버 저장 / 불러오기 실패
- 문의사항 접수 실패
- 관리자 문의 조회 실패
- 주요 페이지 렌더링 오류

### P1 후보

서비스 사용은 가능하지만 핵심 경험에 영향을 주는 문제입니다.

- 포트폴리오 입력 방식이 어렵다는 피드백
- 결과 차트 해석이 어렵다는 피드백
- 저장 / 불러오기 흐름이 직관적이지 않다는 피드백
- Free 플랜 제한이 이해되지 않는다는 피드백
- 모바일에서 특정 화면 사용이 불편하다는 피드백

### P2 후보

문구, 디자인, 편의성 개선 항목입니다.

- 버튼 문구 개선
- 안내 문구 추가
- 차트 제목 / 설명 개선
- FAQ 추가
- 모바일 여백 / 타이포그래피 조정
- 공유 문구 개선

## 베타 공개 판단 기준

아래 조건을 유지하면 Beta v0.1 공개 상태를 유지할 수 있습니다.

- 접속이 정상이다.
- 체험 계정으로 시작할 수 있다.
- Free 플랜에서 포트폴리오 1개가 유지된다.
- 서버 저장 / 불러오기가 정상이다.
- 문의사항이 접수된다.
- 관리자 화면에서 문의를 확인할 수 있다.
- 투자 유의사항이 노출된다.
- 주요 모바일 화면이 깨지지 않는다.

---

## 운영 메모

- 기능 추가보다 피드백 분류가 우선입니다.
- 투자 관련 표현은 수익 보장처럼 보이지 않도록 주의합니다.
- 결제, Pro, 리밸런싱 추천 기능은 법적/정책 검토 후 진행합니다.
- 베타 기간에는 사용자에게 “체험 운영 중”임을 명확히 안내합니다.
