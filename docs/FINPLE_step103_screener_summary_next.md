## 작업 개요

스크리너 하단에 있던 다음 단계 박스를 상단 요약 카드 영역으로 이동했습니다.

## 변경 사항

```text
- 하단 다음 단계 박스 제거
- 상단 요약 카드에 다음 단계 카드 추가
- 현재 추가 대상 / 현재 자산 / 다음 단계 3개 카드 구조 적용
- 다음 단계 카드는 시뮬레이터 이동 버튼 역할 수행
- 미국·한국 자산이 함께 담긴 경우 지표 기준 안내문 표시
```

## 지표 기준 안내

혼합 시장 포트폴리오에서는 아래 유의가 필요합니다.

```text
- 미국 자산 지표는 달러 기준일 수 있음
- 한국 자산 지표는 원화 기준일 수 있음
- BETA는 시장별 기준지수가 다를 수 있음
```
