## 6. BI 컬러 팔레트 기준

현재 `/about` BI 컬러 팔레트는 1행 6열, 원형 버블 스와치 형태입니다. 컬러명은 `Finple` 대소문자 표기를 사용합니다.

| 순서 | 이름 | HEX | 현재 설명 | 권장 역할 |
| --- | --- | --- | --- | --- |
| 1 | Finple Blue | `#38BDF8` | 로고 포인트 컬러 | 브랜드 포인트, 일러스트 강조색, 보조 액션 강조 |
| 2 | Finple Deep Blue | `#2563EB` | 주요 텍스트와 강조 | 강조 텍스트, 링크, 활성 상태, 중요한 수치 강조 |
| 3 | Finple Light Blue | `#E0F2FE` | 부드러운 배경과 박스 | 연한 정보 박스, 섹션 보조 배경, 일러스트 배경 |
| 4 | Finple Navy | `#0F172A` | 주요 버튼과 강조 | 주요 버튼, 강한 텍스트, floating TOP 버튼 |
| 5 | Finple Gray | `#64748B` | 설명문과 보조 정보 | 설명 문구, 보조 라벨, 비활성에 가까운 정보 |
| 6 | Finple White | `#F8FAFC` | 차분한 배경화면 | 페이지 배경, 넓은 화면 배경, 차분한 표면 |

전 라우터 컬러 정리 시 권장 기준:

- 기존 보라색 계열 `#6c63ff`는 브랜드 일러스트나 UI 강조 맥락에서는 우선 `#38BDF8`로 교체 후보입니다.
- 단, SVG 내부의 `#2f2e41`, `#3f3d56`, `#090814` 같은 어두운 구조색은 선, 그림자, 인물 형태일 수 있으므로 일괄 교체하지 않습니다.
- 페이지 전체 배경은 `#F8FAFC`를 기본값으로 보고, 정보성 박스나 부드러운 영역은 `#E0F2FE`를 후보로 봅니다.
- 본문 텍스트와 강한 제목은 `#0F172A`, 설명문은 `#64748B` 기준으로 맞추는 것이 좋습니다.
- 액션 버튼은 `#0F172A`를 기본으로 두고, 브랜드 포인트가 필요한 보조 강조에는 `#38BDF8` 또는 `#2563EB`를 사용합니다.
- 색상 변경은 먼저 라우터별 사용 맥락을 확인한 뒤 진행합니다. 전체 검색 후 무조건 치환하는 방식은 피합니다.

## 7. 레이아웃 기준

`/about` 페이지의 현재 폭 기준:

| 영역 | 기준 |
| --- | --- |
| 히어로 | `max-width: 1180px` |
| 히어로 타이틀/본문 | `max-width: 1120px` |
| 일반 섹션 카드 | `width: min(calc(100% - 64px), 1180px)` |
| BI 카드 | 일반 섹션과 동일하게 `1180px` 기준 |
| 일반 섹션 텍스트/일러스트 | `1.35fr / 0.65fr` |
| BI 상단 텍스트/로고 | 3열 그리드, 텍스트 2열 span, 로고 1열 |
| BI 의미 카드 | 3열 그리드 |
| 컬러 팔레트 | PC 6열, 태블릿 3열, 모바일 2열 |

일러스트 관련:

- PC에서도 모바일에서도 일러스트 이미지 크기는 현 상태를 유지하기로 했습니다.
- 이후 섹션 텍스트 변경이나 카드 폭 변경 후에 필요하면 별도 조정합니다.

## 8. TOP 버튼 기준

`/about`에도 기존 라우터에서 사용하던 floating TOP 버튼을 추가했습니다.

구현:

```jsx
<button
  type="button"
  className="floatingTopButton"
  onClick={() => window.scrollTo({ top: 0, behavior: "smooth" })}
  aria-label="ABOUT 페이지 상단으로 이동"
>
  ↑ TOP
</button>
```

스타일:

- 별도 신규 스타일을 만들지 않고 기존 `src/App.css`의 `.floatingTopButton`을 재사용합니다.
- 다른 라우터의 위치와 서식을 맞추기 위한 목적입니다.
