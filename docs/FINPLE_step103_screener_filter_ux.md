# FINPLE Step 103-9 — Screener Filter UX

## 작업 개요

스크리너 필터를 1차 / 2차 / 3차 구조로 정리했습니다.

기존에는 자산군 필터를 누르면 스타일, 위험도, 초보자 필터가 초기화되어 `배당 + ETF`, `성장 + 개별주` 같은 조합 검색이 어려웠습니다.

이번 단계에서는 필터가 서로 초기화되지 않고 조합되도록 정리했습니다.

## 필터 구조

```text
1차 자산군
- 전체
- ETF
- 개별주

2차 스타일
- 전체
- 초보자
- 대표 ETF
- 배당
- 성장
- 방어
- 공격형

3차 위험도
- 전체 위험
- 낮음~중간
- 중간
- 중간~높음
- 높음
- 매우 높음
```

## 주요 변경

```text
- 자산군 필터 선택 시 스타일 / 위험도 필터를 초기화하지 않음
- 기존 beginnerOnly 체크박스를 스타일 필터의 초보자 항목으로 통합
- 대표 ETF는 ETF + core 후보를 의미하도록 분리
- 결과 영역에 현재 필터 조합을 표시
- PC에서는 자산군 버튼형 유지
- 모바일에서는 자산군 드롭다운 표시
```
