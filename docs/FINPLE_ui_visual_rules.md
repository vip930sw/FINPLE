# FINPLE 공통 화면 시각 규칙

이 문서는 FINPLE의 메뉴, 카드와 섹션을 수정할 때 반복적으로 달라지기 쉬운 색상·간격 기준을 고정하기 위한 공통 인수인계 문서다.

## 1. 밝은 배경의 영문 섹션 라벨

흰색 또는 연한 파란색 배경 위에 표시되는 짧은 영문 라벨은 항상 `Finple Deep Blue`를 사용한다.

- 적용 예: `ROADMAP`, `UPDATES`, `ADMIN MODE`, `PASSWORD`
- 색상: `#2563EB`
- 검은색, Navy `#0F172A`, Gray `#64748B`로 표시하지 않는다.
- 한글 본문 제목과 영문 라벨이 함께 있을 때 영문 라벨은 파란색, 한글 제목은 Navy를 유지한다.
- 어두운 배경 위에서는 가독성을 위한 별도 대비색을 사용할 수 있다.

공통 CSS 기준:

```css
.accountPage .accountHero .sectionLabel,
.accountPage .accountCard .accountMiniLabel {
  color: #2563eb !important;
}
```

특정 화면의 후행 스타일이 공통 규칙을 덮을 가능성이 있으면 해당 화면에서도 같은 색상을 명시한다. `/updates`의 `ROADMAP` 라벨은 `.updatesReleasePage .updateReleaseFooterIntro > .accountMiniLabel`로 한 번 더 고정한다.

## 2. 제목과 설명문의 색상

- 주요 한글 제목: Finple Navy `#0F172A`
- 설명문과 보조 정보: Finple Gray `#64748B`
- 링크, 활성 상태와 중요한 라벨: Finple Deep Blue `#2563EB`
- 부드러운 정보 카드 배경: `#F8FBFF` 또는 Finple Light Blue 계열

영문 라벨과 한글 제목을 동일한 검은색으로 합치지 않는다. 라벨의 파란색이 정보의 계층을 구분하는 역할을 한다.

## 3. 카드 내부 여백

- 외곽 카드의 좌우 폭은 같은 페이지의 주요 콘텐츠 폭과 맞춘다.
- 여러 개의 작은 카드를 외곽 카드 안에 배치할 때 가장자리에 최소 18px의 내부 여백을 둔다.
- 작은 카드 사이 간격은 최소 12px, 권장 14px이다.
- 한 줄 요약 카드도 세로 높이를 지나치게 줄이지 않고 데스크톱 기준 약 96px의 최소 높이를 확보한다.
- 모바일에서는 2열 또는 1열로 전환하고 최소 높이 약 88px을 확보한다.

## 4. 변경 전 확인 사항

- [ ] 밝은 카드의 영문 라벨이 `#2563EB`인가
- [ ] 한글 제목은 Navy, 설명문은 Gray로 계층이 구분되는가
- [ ] 외곽 카드와 내부 카드 사이에 충분한 여백이 있는가
- [ ] 카드 간 간격이 붙어 보이지 않는가
- [ ] 모바일에서도 라벨과 값이 잘리지 않는가
- [ ] 페이지 전용 CSS가 공통 라벨 색상을 덮어쓰지 않는가

새 메뉴나 카드를 추가할 때 이 문서를 먼저 확인하고, 반복되는 시각 규칙은 화면별 임시 수정 대신 공통 선택자에 반영한다.
